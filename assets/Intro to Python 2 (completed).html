# The syntax for odeint() is
# y = odeint(<der func>,<y0>,<x array>)

# <der func> is a defined function that calculates the derivative
# <y0> is the initial y value
# <x array> is an array of x values where we want solution values
# Then odeint() returns an array of numerical solution values at every value from the <x array>
In [21]:
# Let's look at an example.  Suppose we want to solve
# dy/dx = y(3-xy)  on the interval [0,6] where y(0)=1

# First we need to define the derivative function
# It's very important that the function be <der func>(<dep variable>,<indep variable>):
# Otherwise odeint() will get them mixed up
def f(y,x):
    return y*(3-x*y)
In [22]:
# Now that we have the derivative function, all we need to do is create an <x array> and solve!

xx = np.linspace(0,6,100)
yy = odeint(f,1,xx)

    yy = odeint(f,y0,xx)
    plt.plot(xx,yy,label="y0 = "+str(y0), linewidth=3) # str() makes the variable value a string

plt.legend(loc='upper right') # loc='' specifies the location of the legend
plt.axis([0,6,0,15]) # changes the range on the x,y axes
